You are a professional UI/UX designer creating a complete high-fidelity design system and application UI for a modern web application called **Housing Society Hub – Issue & Operations Management System**.

Your task is to design a **fully structured, modern, clean SaaS-style web application interface** with complete navigation, components, and layouts for both **Resident and Admin users**.

The design must be **consistent, professional, responsive, and production-ready**.

---

PROJECT OVERVIEW

Housing Society Hub is a digital platform that helps manage housing society operations such as complaints, notices, maintenance payments, visitor logs, and resident communication.

The application has two main user roles:

1. Resident
2. Admin / Society Management

Each role has its own dashboard and features but shares a consistent design system.

Design rules:
• Navbar background should use #3D4127
• Primary buttons should use #636B2F
• Hover states should use a darker variation of #636B2F
• Card highlights, badges, notifications should use #D4DE95
• Page backgrounds should be light neutral with hints of #BAC095
• Text headings should use #3D4127

Avoid introducing new strong colors outside the palette.
